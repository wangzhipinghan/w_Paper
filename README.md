# w_Paper
文本文件中的字段自由移动与标记

# Features:

1.多栏文字显示，黑板显示；

2.文字块自由移动范围大；

3.使用普通文本文件，不要修改文件内容； (ANSI)

4.操作简单，按按钮提示就可以；

5.文字自由缩放，无边框；

6.简单的思维导向连接和调整无边界，拉伸线增减；

7.背景调整，对比度变化；

8.全窗涂鸦消除，小矩形、小圆圈添加和消除；



# 安装
无需安装，只需解压包文件复制到电脑即可，没有注册残留。

# 注意
移动行或段时，应将鼠标置于绿色右端顶格后拖动。这样的原因是防止重叠字段的误操作。
